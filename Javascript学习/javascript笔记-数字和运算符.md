## 运算符优先级

 JavaScript中的运算符优先级与学校的数学课程相同 - 乘法和除法总是先完成，然后加法和减法（总和总是从左到右进行评估）。

 如果要调整默认运算符优先级，可以把想优先处理的部分用括号括起来。

 `(num2 + num1) / (8 + 2);`

## 递增和递减运算符

 如果想在原有的变量上做加减法，可以使用递增（`++`）或者递减（`--`）。

`guessCount++ = guessCount + 1;`
